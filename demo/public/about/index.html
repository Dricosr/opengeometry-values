This Project

Why opengeometry-values

Every BIM or engineering app eventually runs into the same problem: a number stored without context. No unit discipline, no precision contract, no distinction between what the user typed and what the geometry engine needs. This library is the foundation I kept rebuilding from scratch - so I wrote it once, properly.

Part of the broader OpenGeometry initiative: open-source building blocks for geometry, data, and interoperability in technical engineering software.
